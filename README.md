# AiDocTranslation

## Description
### AiDocTranslation is a system for automated processing and translation of official identity documents such as passports and ID cards.

## Problem
### Manual translation of official documents is time-consuming, error-prone, and inefficient. Small mistakes can lead to serious legal issues.

## Features
- OCR-based data extraction
- AI-powered translation
- Structured data output
- Automatic document formatting
- Reduced human error

## Tech Stack
- Python (OCR & backend)
- TypeScript (frontend)
